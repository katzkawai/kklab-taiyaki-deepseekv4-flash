# 名古屋たい焼き 金鯱堂

名古屋・大須商店街の架空たい焼き店「金鯱堂」の紹介ランディングページです。

> 本サイトはコーディングエージェント **OpenCode + DeepSeek V4 Flash** によって生成されました。
> 生成コスト: **約 $0.01 USD**

## 内容

- 金鯱堂の概要（大須で30年の歴史、店名の由来）
- 3つのこだわり（老舗、手焼き、創作あん）
- 全10種のメニュー（つぶあん・こしあん・カスタード・抹茶あん・チョコレート・さつまいも・白あん・小倉あん・もちもちチーズ・季節限定）
- Google Maps アクセス情報（大須商店街）
